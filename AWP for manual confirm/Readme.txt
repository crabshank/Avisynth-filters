If you specify (file="..."), it will contain the x and y values and the frame number.

The "thresh" parameter is the saturation value at, and under, which the pixel will show up black (default=0.015).

If debug=true:
The "patch" parameter (0-1 [0%-100%]) is the proportion of the image size that the patch which displays the colour that is being turned grey by the white point process will take up on the image.
